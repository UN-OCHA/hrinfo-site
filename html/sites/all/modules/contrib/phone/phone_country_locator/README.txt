Description
-----------
This module is used with the Phone Number module to set the country for the phone field in node/add form to what the user's country is based on IP address.


Prerequisites
-------------
The phone_country_locator.module requires the phone.module and ip2country.module to be installed. Download Ip2Country From http://drupal.org/project/ip2country.

How to Use
-----------
This module adds an option (Enable Country Code Locator) to the Phone Number field settings. You can enable this option after unchecking the (Enable default country code) option.
